Titulo/Cabeçario

<h1></h1>

cria um cabeçario ou titulo para seu conteudo.
tambem existem os sub-titulos e suas hierarquias.(<h2>, <h3> até o <h6>)

Paragrafo

<p></p>

cria um paragrafo no seu conteudo.

Linha horizontal

<hr>

cria uma linha horizontal.

Quebra de linha

<br>

faz uma quebra de linha em um texto.

Imagens

<img src=""alt="">

no src="" pode ser colocado o caminho da imagem caso ela esteja localmente ou a url caso esteja em outro servidor.

no alt="" ficara o texto que sera exibido caso a imagem não carregue

Listas

listas ordenadas

<ol><ol>

listas que possuem uma ordem.
para adicionar um item na lista é usado a tag <li>.
ex:

  1. acordar
  2. tomar café
  3. tomar banho

repare que existe uma ordem.

listas não ordenadas

<ul><ul>

listas que não possuem ordem.
<li> tambem é usado para adicionar um item na lista
ex:
uma lista de compras:

  • arroz
  • café
  • açucar

listas de definição

<dl><dl>

esse tipo de lista como o nome sugere é usado para definir a alguma coisa.
sendo que dentro da tag <dl><dl> devem ter as tags <dt><dt> e <dd><dd>.
sendo que dentro de dt ficara o termo que tera a sua descrição, enquanto no dd ficara a propria descrição do termo.
ex:

        
    <dl>
        <dt>HTML</dt>
        <dd>Linguagem de marcação utilizada para criar o conteudo de sites</dd>
    </dl>

Links

<a href=""></a>

links externos

para adicionar links externos, basta colocar a url do site que você queira colocar dentro do parâmetro href="".

links internos

ja nos links internos, ao envez de uma url, é o nome do arquivo que o link ira apontar, como uma segunda pagina para seu site.

links para dowload

para fazer um link fazer um dowload de algum arquivo, é só colocar o parametro download="" dentro da tag <a> e indicar para o navegador o nome do arquivo dentro do parametro.

imagens responsivas

uma imagem responsiva é uma imagem que se adapta de acordo com o tamanho da tela.
para fazer uma imagem assim usamos a tag <picture></picture> e colocamos as tag <source media="(min-width: )" srcset="" type="image/">.
a tag assusta mas veja bem, dentro do media ficara o tamanho até aonde a imagem vai, se passar, a imagem vai mudar. dentro do srcset vai ficar o caminho da imagem que vai substituir a atual, e dentro do type vai ficar o media-type da imagem.
