Variaveis

São espaços na memoria que servem para guardar dados com um identificador

ex: 
    numero = 5
    palavra = "maça"

tipos de dados

Numeros

Dentro de numeros, temos os numeros inteiros(int), os flutuantes(float) e os complexos(complex)
ex:
numeros inteiros = 1, 2, 3, 4, 5
numeros flutuantes(numeros com virgula) = 1.5, 5.2, 6.02 (note que no python ao envez de "," usamos ".")
numeros complexos = 1j, 5j, 6j

Strings(palavras)

as strings(str) podem ser escritas com aspas simples('') ou aspas duplas("") 

            
            ex: string1 = "Ola, Mundo!"
                string2 = 'Ola, Mundo!'

Booleanos(True ou False)

simplesmente os valores booleanos são valores que vão retornar True ou False, ou seja, sim ou não, verdadeiro ou faso, 0 ou 1
ex: 

            
                numero1 = 10
                numero2 = 11
                numero1 == numero2 #numero1 é igual a numero2?

                False

                numero1 = 5
                numero2 = 5
                numero1 == numero2 #numero1 é igual a numero2?

                True

listas

listas(list) são um identificador para varios dados e para chamar um dado especifico usamos [] e o numero da ordem do dado, do mesmo jeito que usamos para cria-lás tambem
ex: 

            
                lista = [maçã, 9, 3.4, banana, True]

                lista[0] #chamando o dado de ordem 0 
                
                maçã
